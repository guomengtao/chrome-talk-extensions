# Talk-I 文章浏览器

[English](README.md) | [日本語](README_JA.md)

## 描述
Talk-I 是一个 Chrome 扩展程序，提供清晰高效的文章浏览界面。它专注于以列表形式显示文章标题，并在单独的窗口中打开详细内容。

## 功能特点
- 清晰的文章标题列表视图
- 点击打开文章详情窗口
- 按时间筛选文章：
  - 全部文章
  - 今日文章
  - 本周文章
- 实时加载文章
- 响应式弹出窗口

## 开发信息
- 徽章颜色：#0d47a1（深蓝色）
- 角色：文章浏览器
- 存储键名：talk-i-articles
